## 📊**Project Title** :<br/>
Madhav E-Commerce Sales Dashboard (Power BI)<br/>

## 📝 **Project Overview** :<br/>
This project showcases an **interactive E-Commerce Sales Dashboard** built using Power BI.
It provides insights into sales performance, profit trends, customer behavior, and regional distribution, enabling businesses to make data-driven decisions.<br/>

## 🎯 Purpose :
- Analyze overall sales and profit performance  
- Identify top-performing categories and products  
- Understand customer purchasing patterns  
- Track monthly profit trends and losses  
- Support strategic business decisions

 ## 🛠️ Tech Stack : 
- 📊 **Power BI Desktop** – Dashboard development  
- 🔄 **Power Query** – Data cleaning & transformation  
- 📐 **DAX** – Measures & KPI calculations  
- 📁 **Excel / CSV** – Data source

## 📂 Data Source :
- E-commerce dataset including sales, profit, customers, categories, and payment modes  
- Built using a **YouTube Power BI tutorial (project-based learning)**

## 🚀 Features & Highlights:
### 🔍 Business Problem : 
E-commerce businesses often struggle to extract meaningful insights from large datasets, making it difficult to track performance, identify profitable segments, and optimize strategies.

### 🎯 Goal : 
To build an **interactive dashboard** that provides clear insights into sales, profit, and customer behavior for better decision-making.

### 📈 Key Metrics : 
- 💰 Total Sales Amount: **438K**  
- 📈 Total Profit: **37K**  
- 📦 Total Quantity Sold: **5615**  
- 🧾 Average Order Value (AOV): **121K**

### 📊 Key Visuals : 
- 📌 KPI Cards (Sales, Profit, Quantity, AOV)  
- 📉 Profit by Month (Bar Chart)  
- 🗺️ Sales by State  
- 🍩 Quantity by Category (Donut Chart)  
- 📊 Profit by Sub-Category  
- 👥 Sales by Customer Name  
- 💳 Quantity by Payment Mode  
- 🎛️ Quarter-wise Filters (Q1–Q4)

### ⭐ Key Features :
- Interactive and dynamic filtering  
- Multi-dimensional analysis (state, category, customer, payment mode)  
- Profit and loss trend identification  
- Clean and professional dashboard design  
- Business-focused KPIs and storytelling

## 💡 Key Insights : 
-  Clothing category contributes the highest sales (63%)  
-  COD is the most preferred payment mode (44%)  
-  Some months show negative profit (loss periods)  
-  Printers and Bookcases generate the highest profit  
-  Maharashtra leads in overall sales
